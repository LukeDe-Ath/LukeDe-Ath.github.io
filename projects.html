My Projects

Home Assistant

This is an open source home automation operating system that "puts local control and privacy first." The ideal goal is to have working smart lights, switches, plug sockets, and heaters still useable even if we went off the grid (or if the internet just goes out for a while!)

Home Assistant has been a great learning process for me as it has taught me more about Docker containers, reading/edit JSON files, and simply making daily life a bit simpler for the family at home. My dad has solar panels that charge a couple of home batteries. With the help of this automation software, I've been able to maximise the efficiency of green-energy so that it is used as much as possible in the household.

Did you know: Your TV is likely to be the highest energy usage in the house, even on standby it uses a substancial amount of power!
